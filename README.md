# E-commerce-Sales-Analysis-Dashboard
This project analyzes e-commerce sales data using Excel, SQL, Python, and Power BI.

Key Features:
- Data cleaning and preprocessing using Excel and Python (Pandas)
- SQL-based data analysis using aggregation and filtering queries
- Data visualization using Matplotlib (Python)
- Interactive Power BI dashboard with slicers and KPIs

Insights:
- Identified top-performing city and category based on revenue
- Analyzed customer purchasing patterns
- Built interactive dashboard for business decision-making

Tools Used:
Excel | SQL | Python (Pandas, Matplotlib) | Power BI

Outcome:
Developed an end-to-end data analytics project demonstrating data handling, analysis, and visualization skills.
